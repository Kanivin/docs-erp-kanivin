Currency Exchange Settings

KanivinERP comes with 2 predefined configuration for Currency Exchange Rate providers: Frankfurter.app and Exchangerate.host

Frankfurter.app

This is free exchange rate provider. But, it provides exchange rates for a limited set of currencies.
