Google Calendar Integration

KanivinERP provides an integration with Google Calendar in order for all users to synchronize their Google Calendar Events with KanivinERP.

How to set up Google Calendar Integration

In order to allow a synchronization with Google Calendar, you need to authorize KanivinERP to get Calendar Events data from Google. Google Calendar Integration is set up with the following steps:

  • Create OAuth 2.0 Credentials via Google Settings.
  • In the Google Calendar list, click on New. Enter Calendar Name and the User for whom you want to sync and then save it.
  • Depending what data you want to sync, you can select following
    • Pull from Google Calendar - Syncs all event from Google Calendar to KanivinERP.
    • Push to Google Calendar - Syncs all event from KanivinERP to Google Calendar.
  • Now click on Authorize Calendar Access to authorize KanivinERP to get Calendar Events data from Google.
  • Once Authorized, you can manually sync Google Calendar Event or let KanivinERP sync Google Contacts daily.

Creating an Event in KanivinERP
